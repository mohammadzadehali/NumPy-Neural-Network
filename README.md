# NumPy Neural Network

This is a simple neural network that I built from scratch using only NumPy.

I created this project to understand how neural networks work internally, including forward propagation, backpropagation, gradient descent, and updating weights and biases.

## Features

- Forward propagation
- Backpropagation
- ReLU activation function
- Sigmoid activation function
- Mean Squared Error loss
- Gradient descent

## Built With

- Python
- NumPy

## About

This project was made for learning and understanding the basic concepts behind neural networks without using libraries like PyTorch or TensorFlow.
